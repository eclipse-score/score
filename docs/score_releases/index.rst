S-CORE Releases
===============

A **Release** is an officially distributed, versioned, and deployable collection of the software platform. It relates to a collection of features and modules of the SW platform. For further information about the platform releases and module releases see `release documentation <https://eclipse-score.github.io/process_description//main/process_areas/release_management/release_workproducts.html>`_ in the process description.
