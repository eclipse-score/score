Workproducts
############

.. workproduct:: Platform Release Notes
   :id: wp__platform_sw_release_note
   :status: valid
   :tags: release_management

   Release notes describe the qualified SW version including known bugs from own testing and field
   reporting, with clear statement, that these bugs do not lead to violation of any safety
   requirements or with corresponding workaround measures. Platform level.

.. workproduct:: Module Release Notes
   :id: wp__module_sw_release_note
   :status: valid
   :tags: release_management

   Release notes describe the qualified SW version including known bugs from own testing and field reporting, with clear statement, that these bugs do not lead to violation of any safety requirements or with corresponding workaround measures. Module level.

.. workproduct:: Platform Release Plan
   :id: wp__platform_sw_release_plan
   :status: valid
   :tags: release_management

   A release plan for the platform that showcases the roadmap for the upcoming platform releases.
   Indicates which software modules are planned in the upcoming release

.. workproduct:: Module Release Plan
   :id: wp__module_sw_release_plan
   :status: valid
   :tags: release_management

   A release plan for a software module. Indicates which features are included in the software
   module.
